Features

Size Guide

Sky Soles uses the very latest in advanced SOP (Stand on Paper) measurement technologies. Every Sky Soles design is made using different lasts which can vary the sizing. Please ensure that you are using the correct size guide for the style you are interested in. 

 

When printing, ensure that the printer is set to "actual size", as this could vary the size guides advice.
